# Experiments

The repository exposes three minimal starter cases:

- `cases/sphere_sphere`
- `cases/sphere_ellipsoid`
- `cases/block_plane`

Each case is defined as JSON and can be executed through `benchmark_runner` or the Python wrappers in `scripts/`.

The scaffold already exports per-run JSON and can be extended to aggregated CSV, tables, and figures following the protocol described in `prompt/06_EXPERIMENT_COMPLETION_PROTOCOL.md`.
